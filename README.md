Diabetes Prediction Using Machine Learning

Project Overview

This project aims to predict whether a person has diabetes using Machine Learning algorithms.

Dataset Features

- Gender
- Age
- Hypertension
- Heart Disease
- Smoking History
- BMI
- HbA1c Level
- Blood Glucose Level

Target Variable

- Diabetes
  - 0 = No Diabetes
  - 1 = Diabetes

Machine Learning Algorithms Used

1. Decision Tree Classifier
2. Random Forest Classifier

Project Workflow

1. Load the dataset
2. Preprocess the data
3. Encode categorical features
4. Split data into training and testing sets
5. Train Decision Tree model
6. Evaluate model performance
7. Train Random Forest model
8. Compare results

Results

- Decision Tree Accuracy: 97.5%
- Random Forest Accuracy: 98%

Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report

Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

Conclusion

The Random Forest model achieved the highest accuracy of 98% and performed better than the Decision Tree model for diabetes prediction.
